# Brave (Browser)
Download [Brew](https://github.com/BosEriko/brew) as your package manager then install [Brave](https://formulae.brew.sh/cask/brave-browser).

## Install Brave
```sh
brew install --cask brave-browser
```

## Clone the Repository
```sh
mkdir -p ~/Documents/Codes/Configuration
git clone https://github.com/BosEriko/brave.git ~/Documents/Codes/Configuration/brave
```
