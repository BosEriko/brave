# Brave (Browser)
Download [Nix](https://github.com/BosEriko/nix) as your package manager then install [Brave](https://search.nixos.org/packages?query=brave&show=brave).

## Install Brave
```sh
nix-env -iA nixpkgs.brave
```

## Update your settings
Follow this [instruction](instruction.md) to update your settings. Then update your [hotkeys](https://github.com/boseriko/hotkey/blob/main/BROWSER.md) on `brave://settings/system/shortcuts`.
