# Hotkeys
Add the hotkeys [here](brave://settings/system/shortcuts).

## Tab
- New Tab: `ctrl+n`
- Reopen Closed Tab: `ctrl+shift+n`
- Close Tab: `ctrl+w`
- Previous Tab: `ctrl+h`
- Forward: `ctrl+j`
- Back: `ctrl+k`
- Next Tab: `ctrl+l`
## Page
- Reload: `ctrl+r`
- Reload bypassing cache: `ctrl+shift+r`
## Miscellaneous
- Copy URL: `ctrl+b`
